## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')

n = int(input_data[0])  # Number of elements
arr = list(map(int, input_data[1].split()))  # The list of elements
K = int(input_data[2])  # The threshold value

count = 0
for i in range(n):
    for j in range(i + 1, n):
        if arr[i] + arr[j] > K:
            count += 1

print(count)
```
