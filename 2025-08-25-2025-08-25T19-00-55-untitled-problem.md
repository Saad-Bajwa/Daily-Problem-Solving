## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
a = list(map(int, input_data[1].split()))

if n == 0:
    print(0)
else:
    count = 0
    for i in range(n):
        if a[i] == 1:
            count += 1
    print(count)
```
