## Solution

```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
numbers = list(map(int, input_data[1].split()))

average = sum(numbers) / n

for i in range(n):
    if numbers[i] > average:
        print(numbers[i])
```
