## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
arr = list(map(int, input_data[1].split()))
s = int(input_data[2])

# To store the pairs in a list
result = []

# Finding the pairs with the given sum
for i in range(n):
    for j in range(i + 1, n):
        if arr[i] + arr[j] == s:
            result.append((arr[i], arr[j]))

# Print each pair
for pair in result:
    print(pair[0], pair[1])
```
