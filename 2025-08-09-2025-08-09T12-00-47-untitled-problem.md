## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
d = list(map(int, input_data[1].split()))
maximum_sum = -1

for i in range(n):
    for j in range(i + 1, n):
        current_sum = (d[i] - 1) * (d[j] - 1)
        if maximum_sum < current_sum:
            maximum_sum = current_sum

print(maximum_sum)
```
