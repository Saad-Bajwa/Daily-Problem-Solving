## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
elements = list(map(int, input_data[1].split()))
sum_limit = int(input_data[2])

count = 0

# Iterate through all pairs of elements
i = 0
while i < n:
    j = i + 1
    while j < n:
        if elements[i] + elements[j] < sum_limit:
            count += 1
        j += 1
    i += 1

print(count)
```
