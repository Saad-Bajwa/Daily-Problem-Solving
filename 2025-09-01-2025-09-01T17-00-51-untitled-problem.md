## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')

total_cases = int(input_data[0])
results = []

for i in range(1, total_cases + 1):
    n = int(input_data[i])
    if n % 2 == 0:
        results.append('Even: {}\n'.format(n))
    else:
        results.append('Odd: {}\n'.format(n))

print(''.join(results).strip())
```
