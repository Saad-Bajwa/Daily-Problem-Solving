## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
k = int(input_data[0])
numbers = list(map(int, input_data[1].split()))

results = []
for num in numbers:
    if num > k:
        results.append(num)

print(' '.join(map(str, results)))
```
