## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
arr = list(map(int, input_data[1].split()))

for i in range(n):
    for j in range(i + 1, n):
        if arr[i] == arr[j]:
            print(f'Indices: ({i}, {j})')
            break
```
