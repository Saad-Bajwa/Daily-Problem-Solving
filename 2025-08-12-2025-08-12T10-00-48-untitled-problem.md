## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')

test_cases = int(input_data[0])
results = []

for i in range(1, test_cases + 1):
    line = input_data[i]
    a, b = map(int, line.split())
    results.append(a + b)

for result in results:
    print(result)
```
