## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n, k = map(int, input_data[0].split())
arr = list(map(int, input_data[1].split()))

count = 0
for i in range(n):
    for j in range(i + 1, n):
        if arr[i] + arr[j] == k:
            count += 1

print(count)
```
