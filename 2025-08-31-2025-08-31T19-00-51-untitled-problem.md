## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
arr = list(map(int, input_data[1].split()))
target = int(input_data[2])

number_map = {}

for i in range(n):
    complement = target - arr[i]
    if complement in number_map:
        print(f'{number_map[complement]},{i}')
        break
    number_map[arr[i]] = i
```
