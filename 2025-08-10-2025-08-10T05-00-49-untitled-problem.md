## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
p = int(input_data[0])  # number of pairs
results = []

for i in range(1, p + 1):
    a, b = map(int, input_data[i].split())
    if a > b:
        results.append(f'{b} {a}')
    else:
        results.append(f'{a} {b}')

print('\n'.join(results))
```
