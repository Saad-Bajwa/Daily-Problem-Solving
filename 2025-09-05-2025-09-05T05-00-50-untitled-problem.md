## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
a = list(map(int, input_data[1].split()))

max_value = -1

for i in range(n):
    if a[i] > max_value:
        max_value = a[i]

print(max_value)
```
