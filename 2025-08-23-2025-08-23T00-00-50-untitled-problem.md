## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')

t = int(input_data[0])

results = []
for i in range(1, t + 1):
    a, b = map(int, input_data[i].split())
    if a + b > 0:
        results.append('Positive')
    elif a + b < 0:
        results.append('Negative')
    else:
        results.append('Zero')

print('\n'.join(results))
```
