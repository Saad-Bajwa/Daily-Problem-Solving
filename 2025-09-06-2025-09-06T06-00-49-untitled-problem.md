## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
a = list(map(int, input_data[1].split()))
k = int(input_data[2])

# Function to find all pairs with a sum equal to k
pairs = []
for i in range(n):
    for j in range(i+1, n):
        if a[i] + a[j] == k:
            pairs.append((a[i], a[j]))

# Printing the pairs
for p in pairs:
    print(p)
```
