## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
s = list(map(int, input_data[1].split()))

# Calculate the maximum sum of the non-adjacent numbers
if n == 0:
    print(0)
elif n == 1:
    print(s[0])
else:
    prev = s[0]
    curr = max(s[0], s[1])
    for i in range(2, n):
        new_curr = max(curr, prev + s[i])
        prev = curr
        curr = new_curr
    print(curr)
```
