## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
a = list(map(int, input_data[1].split()))

max_so_far = a[0]
max_end_here = a[0]

for i in range(1, n):
    max_end_here = max(a[i], max_end_here + a[i])
    max_so_far = max(max_so_far, max_end_here)

print(max_so_far)
```
