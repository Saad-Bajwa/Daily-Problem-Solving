## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')

# Read the input
n = int(input_data[0])  # number of elements
arr = list(map(int, input_data[1].split()))  # array elements

# Sort the array
arr.sort()
count = 0

# Count pairs with sum equal to k
for i in range(n):
    for j in range(i + 1, n):
        if arr[i] + arr[j] == 0:
            count += 1

# Output the result
print(count)
```
