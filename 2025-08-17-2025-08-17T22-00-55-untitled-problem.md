## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
numbers = list(map(int, input_data[1].split()))

result = []
for i in range(n):
    if numbers[i] % 2 == 0:
        result.append(numbers[i])

print(' '.join(map(str, result)))
```
