## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
d = int(input_data[1])
a = list(map(int, input_data[2].split()))

# Initialize count of pairs
count_pairs = 0

# Count valid pairs
for i in range(n):
    for j in range(i + 1, n):
        if (a[i] + a[j]) % d == 0:
            count_pairs += 1

print(count_pairs)
```
