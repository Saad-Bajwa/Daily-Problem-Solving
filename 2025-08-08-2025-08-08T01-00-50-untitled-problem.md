## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
arr = list(map(int, input_data[1].split()))
result = [''] * n

for i in range(n):
    result[i] = '\033[32m' + str(arr[i]) + '\033[0m' if arr[i] % 2 == 0 else '\033[31m' + str(arr[i]) + '\033[0m'

print(' '.join(result))
```
