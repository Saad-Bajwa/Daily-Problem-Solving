## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
sequence = list(map(int, input_data[1].split()))

# Find the longest increasing subsequence using dynamic programming
if n == 0:
    print(0)
else:
    dp = [1] * n
    for i in range(1, n):
        for j in range(i):
            if sequence[i] > sequence[j]:
                dp[i] = max(dp[i], dp[j] + 1)
    print(max(dp))
```
