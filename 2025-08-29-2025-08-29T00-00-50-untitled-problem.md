## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')

t = int(input_data[0])
results = []

for i in range(1, t + 1):
    a, b = map(int, input_data[i].split())
    results.append(a + b)

for result in results:
    print(result)
```
