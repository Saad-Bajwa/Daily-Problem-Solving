## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')

n = int(input_data[0])
arr = list(map(int, input_data[1].split()))

if n == 0:
    print(0)
else:
    arr.sort()  
    distinct_count = 1
    for i in range(1, n):
        if arr[i] != arr[i - 1]:
            distinct_count += 1
    print(distinct_count)
```
