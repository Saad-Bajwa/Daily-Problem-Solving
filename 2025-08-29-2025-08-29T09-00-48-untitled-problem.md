## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
m = int(input_data[1])
arr = list(map(int, input_data[2].split()))

count = 0

# Count how many pairs (i, j) where arr[i] + arr[j] == m
for i in range(n):
    for j in range(i + 1, n):
        if arr[i] + arr[j] == m:
            count += 1

print(count)
```
