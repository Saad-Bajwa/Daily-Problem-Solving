## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
a = list(map(int, input_data[1].split()))
p = int(input_data[2])

# Initialize the result variable for storing count of pairs
count = 0

# Using a set to keep track of seen numbers
seen = set()

for number in a:
    complement = p - number
    if complement in seen:
        count += 1
    seen.add(number)

print(count)
```
