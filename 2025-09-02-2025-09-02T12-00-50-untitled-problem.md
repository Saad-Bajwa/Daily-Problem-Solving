## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
arr = list(map(int, input_data[1].split()))

max_sum = float('-inf')

for i in range(n):
    for j in range(i, n):
        current_sum = sum(arr[i:j+1])
        if current_sum > max_sum:
            max_sum = current_sum

print(max_sum)
```
