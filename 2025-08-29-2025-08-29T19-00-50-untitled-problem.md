## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
requests = [tuple(map(int, line.split())) for line in input_data[1:n+1]]

answers = []

for (s, t) in requests:
    answers.append(s * t)

for answer in answers:
    print(answer)
```
