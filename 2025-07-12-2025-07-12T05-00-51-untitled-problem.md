## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
elements = list(map(int, input_data[1].split()))

max_product = float('-inf')

for i in range(n):
    for j in range(i + 1, n):
        product = elements[i] * elements[j]
        if product > max_product:
            max_product = product

print(max_product)
```
