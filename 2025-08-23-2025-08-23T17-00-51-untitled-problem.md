## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
numbers = list(map(int, input_data[1].split()))

result = []

for i in range(n):
    for j in range(i + 1, n):
        if numbers[i] + numbers[j] == 0:
            result.append((numbers[i], numbers[j]))

for pair in result:
    print(pair[0], pair[1])
```
