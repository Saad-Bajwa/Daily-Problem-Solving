## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
s = list(map(int, input_data[1].split()))

# Initialize the count of pairs
count = 0

# Use a nested loop to count pairs
for i in range(n):
    for j in range(i + 1, n):
        if s[i] == s[j]:
            count += 1

print(count)
```
