## Solution
```python
import sys

input_data = sys.stdin.read().strip().split('\n')
n = int(input_data[0])
arr = list(map(int, input_data[1].split()))
target = int(input_data[2])

indexes = []
for i in range(n):
    if arr[i] == target:
        indexes.append(i)

# Output the indexes as a comma-separated string
print(','.join(map(str, indexes)))
```
